# SirtetJS

GameBoy Tetris implemented in Phaser.

* Press any key on the start screen to enter the menu. (But where's the any key?)
* You can toggle between 1p and 2p on the menu screen, but 2p isn't supported. Use `space` when selecting 1 Player to advance to the game.
* C and X rotate clockwise and counterclockwise, respectively.
